AUTOTORINO AI COPILOT
=====================

INSTRUCTIONS:
1. Windows Users: Double-click "Start_Autotorino.bat"
2. Mac Users: Double-click "Start_Mac.command"

FIRST RUN NOTE:
The first time you run this, it will download about 10GB of data (AI Engine + Model). 
Please be patient, it may take 10-20 minutes depending on your internet.

PERFORMANCE NOTE:
- If you have an NVIDIA RTX card or Mac M1/M2/M3: The AI will be fast.
- If you have a standard office laptop (Intel/AMD graphics): The AI will work, but it will be slow. This is normal.

TROUBLESHOOTING:
If the window closes immediately, ensure you have Python 3.10 or 3.11 installed.
